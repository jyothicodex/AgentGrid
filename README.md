# AgentGrid OS 🚀

**AgentGrid** is your AI Chief of Staff. It is a multi-agent operating system designed to act as your executive team, coordinating tasks across different domains such as Product, Finance, Marketing, and Hiring.

## Overview
Behind the elegant UI, AgentGrid simulates an autonomous AI workflow where specialized agents operate in tandem. 
*   **CEO Agent:** Orchestrates cross-functional goals and MVP architecture.
*   **Finance Agent:** Manages runway, SaaS costs, and budget planning.
*   **Hiring Agent:** Actively sources candidates, screens resumes, and builds the team.
*   **Marketing Agent:** Drafts GTM strategies and campaigns.

## Features
- **Live Agent Dashboard:** Monitor the health of your startup and see real-time active statuses of your AI workforce.
- **Execution Roadmap:** An auto-updating path to launch based on agent execution.
- **Agent Directory:** Configure and chat with your individual specialized agents.

## Tech Stack
- **Frontend:** React, Vite, Tailwind CSS, Lucide Icons
- **Backend:** Python, FastAPI (Mockup for LangGraph orchestration)

## Getting Started
To run the UI locally:
1. Navigate to the `frontend` directory.
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`
